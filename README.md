### 👋 About me

I'm a senior full-stack engineer with **7+ years** building Angular and React applications at enterprise scale, with hands-on backend experience in Java and Spring Boot. Most recently I spent **5.5 years at General Motors** as a Frontend Lead — working across six production applications and owning frontend architecture on three of them, from a two-year legacy platform rebuild to executive dashboards powered by Databricks and an LLM-powered search feature built on Glean's Web SDK.
